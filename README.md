# AI Content Generator V2

A full-stack AI-powered content generation platform built using the MERN stack. The application allows users to securely generate AI-powered content while enforcing server-side usage limits through a trial subscription system.

## Features

- 🤖 AI-powered content generation using the OpenAI API
- 🔐 Secure user authentication with JWT and bcrypt password hashing
- 👤 User registration and login system
- 📊 Trial subscription model with per-user generation limits
- 🛡️ Server-side validation and protected API routes
- ☁️ MongoDB Atlas for cloud database storage
- 🏗️ MVC architecture for a scalable and maintainable backend
- ⚡ Responsive React frontend with RESTful API integration

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- OpenAI API
- JWT Authentication
- bcrypt
- REST APIs

## Key Highlights

- Developed a complete MERN-stack application with a modular MVC backend.
- Integrated the OpenAI API to generate AI-powered content dynamically.
- Implemented JWT-based authentication and bcrypt password hashing for secure user management.
- Designed a server-enforced trial subscription system with per-user generation limits stored in MongoDB.
- Built protected API endpoints to prevent unauthorized access to content generation features.
- Structured the project for scalability, maintainability, and easy future feature additions.
